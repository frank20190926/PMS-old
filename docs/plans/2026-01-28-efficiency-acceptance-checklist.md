# 人效中心验收清单（可执行版）

> 覆盖核心链路 + 老系统回写联动

## A. 项目初始化与排期
- [ ] 新项目进入人效中心后状态为“未排期”
- [ ] 项目详情页“初始化/排期”可用
- [ ] 项目列表“快速初始化”可用
- [ ] 模板选择 → 自动排期 → PM 调整 → 保存成功
- [ ] 保存后生成任务并可下发

## B. 任务与进度
- [ ] 任务层级：阶段→里程碑→周/日任务显示正确
- [ ] 状态流转：PENDING→IN_PROGRESS→DELAYED→COMPLETED
- [ ] 自动延期：到期未完成自动标记 DELAYED
- [ ] 手动延期：PM 可标记/取消
- [ ] 任务完成需 PM 验收确认
- [ ] 任务进度可回退（PM 操作）
- [ ] 里程碑/阶段进度可手动维护
- [ ] 建议进度按工时加权展示

## C. 日报
- [ ] 工作日必须提交日报（限制生效）
- [ ] 日报仅能关联已下发任务
- [ ] 日报提交不更新进度
- [ ] PM 审核后更新任务进度
- [ ] 未提交日报会记录缺失并通知 PM
- [ ] 日报附件：上传/查看/删除权限生效
- [ ] 工时 > 8 小时允许提交但需理由

## D. 周报
- [ ] 工作周（周一-周五）生成草稿
- [ ] 员工可编辑补充并提交
- [ ] 周五截止规则生效
- [ ] 个人总览 + 项目拆分视图可用
- [ ] 缺失日报天数在周报中标记

## E. 老系统回写联动
- [ ] 提交日报即创建老系统日报
- [ ] 审批通过更新同一条老系统日报
- [ ] 按项目拆分回写
- [ ] 回写工时固定 8 小时（超出仅留人效）
- [ ] 同步审计表记录成功/失败/重试
- [ ] 失败通知 + 人工重试入口可用

## F. 权限与数据范围（抽样）
- [ ] 员工仅可见自身任务/日报/周报
- [ ] PM 可管理任务、审批日报、调整进度
- [ ] 部门负责人只读查看范围受限
- [ ] 跨部门可见范围由 PM 配置生效

## G. 统计与报表
- [ ] 延期统计：自动+手动，排除归档无工时
- [ ] 工时对比：任务口径 + 里程碑口径
- [ ] 成员工时排行：周/月切换

## H. 数据对账（必须）
- [ ] 人效日报与老系统 `pms_report` 逐条对齐
- [ ] 对账项：项目ID、日期、工时、内容、起止时间

---

**通过标准**：
- 上述清单全部通过
- 对账一致
- 权限抽样通过
