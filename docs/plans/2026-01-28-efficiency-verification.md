# 人效中心核心链路验收清单

**日期**: 2026-01-28
**范围**: 人效中心核心链路（任务/日报/进度/周报/回写）

---

## 1. 日报流程
- [ ] 员工仅能对已下发任务填报日报
- [ ] 当日提交日报后，状态为 SUBMITTED
- [ ] PM 审核通过后日报状态为 APPROVED
- [ ] 审核通过后任务进度更新（支持回退）
- [ ] 未提交日报能标记缺失并触发通知（如有通知功能）

## 2. 日报回写（老系统 pms_report）
- [ ] 提交日报后生成 eff_sync_audit 记录（按项目拆分）
- [ ] 同一日报+项目重复提交不重复生成（幂等校验）
- [ ] 审核通过后更新老系统日报内容
- [ ] 回写失败记录状态为 FAILED，包含错误信息与下次重试时间
- [ ] 手动重试接口可触发重新回写

## 3. 周报流程
- [ ] 周报范围为周一至周五
- [ ] 周报自动聚合日报：总结/成果/问题/下周计划
- [ ] 提交即生效（不需要审核）
- [ ] 周五前可提交（前端提示存在）

## 4. 进度管理
- [ ] 任务列表展示“手动进度”和“建议进度”
- [ ] 建议进度根据开始/结束日期动态计算
- [ ] 任务状态自动流转提示（到期/开始）

## 5. 统计与对账（如启用）
- [ ] 工时统计与日报总工时一致
- [ ] 项目维度工时汇总与日报拆分一致
- [ ] 回写日志与审计表记录可对账
